# My Portfolio — Starter

Đây là template tĩnh cho một website portfolio + blog đơn giản.

## Nội dung đã có
- `index.html` — trang chính (single-file, dễ chỉnh sửa)
- `posts/2025-12-25-hello.html` — bài mẫu
- `README.md`, `DEPLOY.md`, `LICENSE`, `.gitignore`

## Hướng dẫn nhanh (push lên GitHub)
1. (Tùy chọn) Đổi tên repo nếu muốn dùng **GitHub Pages user site**: đặt tên repo là `username.github.io`.
2. Khởi tạo git và commit:

```bash
git init
git add .
git commit -m "Initial commit: portfolio starter"
```

3. Tạo repo trên GitHub (web UI) hoặc dùng `gh` CLI:

```bash
gh repo create username/my-portfolio --public --source=. --remote=origin
```

4. Push lên GitHub:

```bash
git push -u origin main
```

5. Bật GitHub Pages (Settings → Pages) hoặc dùng `gh` to enable pages for a `username.github.io` repo.


## Muốn tôi làm tiếp
- Tôi có thể **khởi tạo git và commit** file này cho bạn (cần xác nhận). 
- Tôi cũng có thể **hướng dẫn push và bật GitHub Pages**, hoặc tự động tạo repo trên GitHub (nếu bạn cung cấp quyền/token/gh CLI authenticated session).
